Conectar base de datos

Solo se hace una vez. Tarda 3 minutos.

01 Entrá a jsonbin.io y creá una cuenta gratuita.
02 Hacé clic en CREATE BIN. En el campo de contenido pegá exactamente: {"articles":[],"stats":{}} — luego CREATE.
03 Copiá el BIN ID que aparece en la URL o arriba del bin (24 caracteres).
04 En el menú de tu cuenta → API Keys → copiá la X-Master-Key.
05 Pegá ambos datos acá abajo y guardá.
